# **App Name**: Code Duel

## Core Features:

- Category Selection: Allow users to choose between 'First Year' and '2nd/3rd/4th Year' difficulty levels, routing them to different question sets and challenges.
- Instructions Page: Display challenge rules, time limit, and scoring details before the challenge begins.
- Timed Challenge: Start a 10-minute timer upon clicking 'Start Challenge.'
- Question Generator: Generate a random question, using different question types and formats for each difficulty level.
- Answer Validation: Validate user answers against the correct output or input and give 10 points for a correct response.
- Score Calculation: Calculate total score, number of correct answers, and provide performance feedback at the end of the challenge. Also, implement a results auto-submit feature when the timer hits zero.
- Admin Panel: Simple interface for adding, editing questions, and setting marks. Consider adding a generative AI feature tool for helping come up with questions. If the generated question is ambiguous or does not fit, provide an informative message why.

## Style Guidelines:

- Primary color: Dark blue (#1A237E) to convey a professional and serious tone.
- Background color: Light gray (#F5F5F5) to provide a clean and modern backdrop.
- Accent color: Yellow (#FFEB3B) to highlight important elements like buttons and timers.
- Body and headline font: 'Inter' (sans-serif) for clear, modern readability.
- Use minimalistic coding-themed icons to enhance user experience without cluttering the interface.
- Ensure a responsive layout that adapts seamlessly to different screen sizes.
- Implement subtle animations on button hover and submission for improved user feedback.
