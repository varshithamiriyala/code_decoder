# 🧩 Code Decoder Game

A fun game where players decode hidden codes or messages using logic and clues.

## 🚀 Features
- Code guessing gameplay  
- Hints and clues system  
- Input validation  
- Interactive UI  

## 🛠️ Tech Used
- HTML  
- CSS  
- JavaScript  

## ▶️ Run
Open `index.html` in a browser.
